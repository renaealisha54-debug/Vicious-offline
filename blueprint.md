# **App Name**: Vicious Offline

## Core Features:

- Intelligent Text Workspace: A streamlined editor interface supporting Markdown and plain text with auto-saving to local browser storage.
- AI Content Assistant Tool: A feature that allows users to select text and use reasoning to either summarize notes or explain complex code snippets via a tool.
- Language Logic Engine: Automatically detects the programming language or text type using an AI-powered classifier to apply appropriate formatting.
- File Portability Suite: Native import and export capabilities allowing users to save files as .txt or .md directly to their device.
- Document Revision Manager: Simple session-based history allowing users to revert changes or clear the workspace for a fresh start.
- Draft Snapshot System: Persist multiple snippets or documents within the local browser environment without needing an account.

## Style Guidelines:

- Primary color: Electric Periwinkle (#9494FF) used for focus states and primary actions, conveying modern efficiency.
- Background: Deep Obsidian (#0B0B13) providing a high-contrast, eye-strain-reducing environment for long writing sessions.
- Accent: Cool Azure (#7EB1FF) for status indicators and tool notifications to maintain an analogous palette hierarchy.
- Headline font: 'Space Grotesk' for a technical, scientific aesthetic in headers.
- Body font: 'Inter' for clear, objective functional labels.
- Code font: 'Source Code Pro' specifically for the editor workspace to ensure optimal readability for text and code.
- A focused, single-column workspace layout with floating controls to maximize content real estate while maintaining an 'agency portfolio' minimalism.
- Liquid-smooth transitions for mode switching (Editor to Preview) and subtle fades during the auto-save indicator trigger.
